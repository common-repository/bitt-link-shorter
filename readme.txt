WP Simple Plugin Get free link shortener plugin for your wordpress.

== Description ==

Bitt Link Shortener plugin is a powerful, extremely scalable and light weight url shortener tool introduced by Seerox.

It concise long urls into small ones. So user can easily share it on social media or email to anyone. The ultimate benefit of this plugin is that these short urls are easy to remember and doesn't look spammy. Brand are easily recognised by these short links which enhances your web marketing outcomes.

It is based on its own Bitt link Engines API and database clusters so your WP database and server will not be over loaded. You will be able to create as many links as you want.

This plugin has its great use while setting up a new site, or if do you need to go back and forth with uploading Plugins from your local computer.

== Installation ==


= Minimum Requirements =

* WordPress 3.8 or greater
* PHP version 5.2.4 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Bitt Link Shortener, log in to your WordPress dashboard, navigate to the Plugins menu and click Add Bulk Link.

In the search field type “Bitt Link Shortener” and click Search Plugins. Once you’ve found our simple plugin upload plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our Bitt Link Shortener plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work when you will update the plugin.
